# Code Foundations (Go)

A structured, long-term project for mastering programming fundamentals, data structures, algorithms, and systems thinking — implemented entirely in Go.

## Structure

- `basics/` — basic syntax, control flow, pointers, functions
- `dsa/` — data structures implemented from scratch
- `algorithms/` — sorting, searching, graph traversals, recursion
- `experiments/` — scratch code, tests, explorations
